시맨틱 웹

기계적인 검색엔진은 어떠한 태그가 어떠한 기능을 하는지 분별할 수 없고 웹페이지에서 데이터를 효율적으로 추출할 수 없다. 특정한 태그에 의미를 부여해서 웹 페이지를 만들자는 시도가 시작됨.

HTML5 시맨틱 구조 태그

header : 헤더를 의미
nav : 네비게이션 구현시 (메뉴처럼 생각해도 됨)
aside : 사이드에 위치하는 공간
section : 여러 중심 내용을 감싸는 공간
article : 글자가 많은 부분
footer : 푸터
